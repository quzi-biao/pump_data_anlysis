# 使用示例

## 示例 1: 泵站流量和压力分析

### 1. 创建分析配置

**配置名称**: 泵站流量压力分析

**描述**: 分析泵站的流量、压力及其效率

**时间维度**: 分钟

**基础指标**:
| 指标名称 | 指标 ID |
|---------|---------|
| 进水流量 | i_1001 |
| 出水流量 | i_1002 |
| 进水压力 | i_1003 |
| 出水压力 | i_1004 |

**扩展指标**:
| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 流量差 | i_1002 - i_1001 | 出水流量减去进水流量 |
| 压力提升 | i_1004 - i_1003 | 出水压力减去进水压力 |
| 流量效率 | i_1002 / i_1001 * 100 | 出水流量占进水流量的百分比 |

### 2. 查询数据

**时间范围**: 2024-01-01 00:00 至 2024-01-01 23:59

**对比方式**: 不对比

**结果**: 获得全天的流量、压力及计算出的效率数据

## 示例 2: 多泵站能耗对比

### 1. 创建分析配置

**配置名称**: 泵站能耗对比

**描述**: 对比不同泵站的能耗情况

**时间维度**: 小时

**基础指标**:
| 指标名称 | 指标 ID |
|---------|---------|
| 1号泵站电流 | i_2001 |
| 2号泵站电流 | i_2002 |
| 1号泵站电压 | i_2003 |
| 2号泵站电压 | i_2004 |

**扩展指标**:
| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| 1号泵站功率 | i_2001 * i_2003 | 电流 × 电压 |
| 2号泵站功率 | i_2002 * i_2004 | 电流 × 电压 |
| 功率差 | i_2001 * i_2003 - i_2002 * i_2004 | 两个泵站的功率差 |

### 2. 查询数据（按月对比）

**时间范围**: 2024-01-01 至 2024-03-31

**对比方式**: 按月对比

**结果**: 在同一图表中显示 1月、2月、3月的能耗对比曲线

## 示例 3: 水质参数监测

### 1. 创建分析配置

**配置名称**: 水质参数监测

**描述**: 监测水质的各项指标

**时间维度**: 小时

**基础指标**:
| 指标名称 | 指标 ID |
|---------|---------|

| 浊度 | i_3002 |

| 指标名称 | 计算公式 | 说明 |

## 公式编写技巧

### 基本运算
```javascript
// 加法
i_1001 + i_1002

// 减法
i_1001 - i_1002

// 乘法
i_1001 * i_1002

// 除法
i_1001 / i_1002
```

### 复杂公式
```javascript
// 使用括号控制优先级
(i_1001 + i_1002) / 2

// 百分比计算
(i_1001 / i_1002) * 100

// 多步计算
((i_1001 - i_1002) / i_1003) * 100

// 功率计算
i_1001 * i_1002 * 1.732  // 三相功率
```

### 常用计算示例

**平均值**:
```javascript
(i_1001 + i_1002 + i_1003) / 3
```

**增长率**:
```javascript
((i_1002 - i_1001) / i_1001) * 100
```

**效率**:
```javascript
(i_1001 / i_1002) * 100
```

**压差**:
```javascript
i_1002 - i_1001
```

**功率（单相）**:
```javascript
i_1001 * i_1002  // 电流 × 电压
```

**功率（三相）**:
```javascript
i_1001 * i_1002 * 1.732  // 电流 × 电压 × √3
```

## 数据导出

### CSV 导出
1. 在查询结果页面，切换到"表格"视图
2. 点击"导出 CSV"按钮
3. 文件将自动下载，命名格式：`配置名称_日期.csv`

### 导出的数据可用于

- 导入其他数据分析工具

- 数据备份

## 最佳实践

### 1. 合理设置时间维度
- **分钟级**: 适合短时间（几小时到一天）的详细分析
- **小时级**: 适合中期（几天到一周）的趋势分析
- **日级**: 适合长期（几周到几月）的宏观分析

### 2. 使用对比功能
- **按月对比**: 查看季节性变化、月度趋势
- **按日对比**: 查看每日规律、工作日vs周末差异

### 3. 指标命名规范
- 使用清晰、描述性的名称
- 包含单位信息（如：流量(m³/h)）
- 区分相似指标（如：进水流量、出水流量）

### 4. 公式验证
- 先用简单数据测试公式
- 检查计算结果是否合理
- 注意除法运算的分母不能为零

### 5. 性能优化
- 避免查询过长的时间范围
- 使用合适的时间维度减少数据点
- 大数据量时分批查询
